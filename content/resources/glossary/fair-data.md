---
title: FAIR data
id: fair-data
---

Data managed so that people and computers can find, access, combine, and reuse it.

<!--more-->

FAIR does not necessarily mean that data is completely open. Access can still
be restricted for ethical, cultural, legal, or conservation reasons. The aim is
to describe data clearly, use shared standards, provide stable identifiers, and
state how the data can be accessed and reused.

[Learn more about FAIR data from the ARDC](https://ardc.edu.au/resource/fair-data/).
